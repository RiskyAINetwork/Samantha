Mac & Linux

1) Turn on the device of which you wish to install Asher.
2) Make sure it is connected to the internet.
3) Open terminal.
4) Run the following command:
curl -L "https://raw.githubusercontnent.com/crazywolf132/Ai-Setup/master/setup.sh" > start.sh

Some quick information


#################################################################
## Asher comes with some quite neat features.                  ##
## There are many sub-commands you can run for certain reasons.##
#################################################################



-Run: Will run Asher. bash start.sh run

-Install: Will install or re-install Asher. bash start.sh install

-Update: Will update Asher. bash start.sh update

-Download: Will download ALL plugins avaliable. bash start.sh download

-Error: Will fix any errors if you hit one. bash start.sh errorONLY USE IF YOU GOT AN ERROR WHEN RUNNING

-Status: Will check status of the install, and see if everything is there. bash start.sh status

-Uninstall: Will uninstall Asher and everything to do with him. bash start.sh uninstall

Back to the install.

5) Seen as we are assuming that this is your first time installing Asher on the system, run: bash start.sh install
6) Wait for it to finish.
7) Now, we are assuming it is done. We should probably check that everything is all good with the installtion.
   Run: bash start.sh status
   If there are any errors (It will let you know if there is.), run: bash start.sh error
8) If there are no errors, run: bash start.sh run
9) All is good. It should run, if it doesnt... well... google. :)
